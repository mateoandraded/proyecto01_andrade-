# Portafolio - Mateo Andrade

Repositorio del desarrollador (Proyecto 01) para el curso de Desarrollo de Aplicaciones Web y Moviles.

Portafolio hecho con HTML, Tailwind CSS (CDN) y un poco de JavaScript, basado en una plantilla de Tailwind.

## Estructura

- `index.html` - pagina principal
- `css/styles.css` - fuente y estilos extra
- `js/main.js` - fecha de actualizacion

## Contenido

- Presentacion y datos del estudiante
- Proyectos con sus tecnologias y enlaces a los repositorios
- Tecnologias aprendidas en el curso
- Contacto

## Despliegue

Publicado con GitHub Pages desde la rama `main`.
